# Network 01 - Pcap

In questa serie di challenge verranno introdotte alcune tematiche necessarie per risolvere le challenge relative alla network security.

Per risolvere questa challenge è necessario comprendere cosa è un file pcap e come analizzarlo.

Tra i vari modi per analizzare i file pcap, un ottimo software consigliato con interfaccia grafica è [Wireshark](https://www.wireshark.org/).

Per installare Wireshark su linux è sufficiente installare il pacchetto dalle standard repository (e.g. sudo apt install wireshark).

Al seguente [link](https://www.wireshark.org/download.html) sono disponibili invece le versioni per Windows e macOS.

La flag di questa challenge si trova all'interno del pacchetto TCP successivo all'handshake.
